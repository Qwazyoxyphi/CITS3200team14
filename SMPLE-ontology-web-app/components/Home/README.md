### TheHome

This component encapsulates all the logic that makes up the `Home Page`. 

#### Usage
This component should be called in the `pages/index.vue` directory which is the `home page`
```html
<template>
  <div class="wrapper">

    <TheHome />

  </div>
</template>
```
